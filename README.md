# clone1-tabnews

Implementação simples inspirada em http://www.tabnews.com.br — projeto para estudo e demonstração.

**Visão geral**
- Projeto minimalista em Python que demonstra integração com `streamlit` e manipulação de dados com `pandas`.

**Funcionalidades**
- Exemplo de app web usando `streamlit`.

**Pré-requisitos**
- Python 3.8+ instalado
- `pip` disponível

**Instalação (local)**
1. Criar e ativar um ambiente virtual (recomendado):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Instalar dependências:

```powershell
pip install -r requirements.txt
```

**Executar**

Se for um app `streamlit` (arquivo `app.py`):

```powershell
streamlit run app.py
```

**CI**
O repositório inclui um workflow do GitHub Actions (`.github/workflows/python-ci.yml`) que instala as dependências e faz um teste de importação básica.
